# 🧑‍💻 Shashank Swaraj — Personal Portfolio

A handcrafted personal portfolio website built with vanilla HTML, CSS, and 
JavaScript. Designed to showcase my projects, skills, and research work 
as a CS & Engineering student.

🌐 Live at: [your-github-pages-link-here]

---

## 📌 Features

- 🖱️ Custom animated cursor with smooth lag-follow ring effect
- ⌨️ Typewriter animation on hero section tag
- 🎞️ Scroll-triggered reveal animations (IntersectionObserver)
- 📌 Sticky navbar with active link highlight on scroll
- 🖼️ Clickable expandable profile picture in navbar
- 🌐 Animated CSS grid background in hero section
- 📱 Fully responsive layout (mobile + desktop)
- 🎨 Warm coffee-tone color palette over dark base
- 🖱️ Custom scrollbar styling

## 💡 JavaScript Concepts Used

| Concept | Where Used |
|---|---|
| `IntersectionObserver` | Scroll reveal animations |
| `requestAnimationFrame` | Smooth cursor ring follow |
| `setInterval` + `slice` | Typewriter effect |
| `e.stopPropagation()` | Profile pic expand/close |
| `classList.toggle/remove` | State management, no libraries |
| `window.addEventListener('scroll')` | Active nav link tracking |
